Crawler
Agentic VS Code extension that:

Scans your workspace for issues using multiple models
Lets you configure local and API providers
Supports swarm mode (multi-model merge)
Generates unified diffs to fix issues
Applies patches to files
Shows issues and providers in a sidebar
Benchmarks providers for latency
Configure providers via:

Commands:
Crawler: Add Local Provider
Crawler: Add API Provider
Crawler: Benchmark Providers
Settings:
crawler.providers
